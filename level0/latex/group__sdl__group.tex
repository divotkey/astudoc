\doxysubsection{Detailed Description}
Functions to write (simple) S\+D\+L-\/based applications. 

Simple Direct\+Media Layer (S\+DL) is a cross-\/platform development library providing low level access to audio, keyboard, mouse, joystick, and graphics hardware. The developer versions of the S\+DL library must be installed separately, although the procedure differs for the different operating systems. The developer versions of the S\+DL library are available here\+: \href{https://www.libsdl.org/}{\texttt{ https\+://www.\+libsdl.\+org/}}\hypertarget{group__sdl__group_sdl_example_sect}{}\doxysubsection{Minimal S\+D\+L-\/based Application}\label{group__sdl__group_sdl_example_sect}
