Detailed Description

This module offers functions to graphical operations.

It offers functions to load, store, create and manipulate images. Currently only BMP files are supported.

◆ CreateImage()

int CreateImage ( int  w,
int  h 
)

Creates a new image with the specified width and height. The new image is created internally and can be accessed using other functions within this module. Any image which might have been created or loaded before is deleted by calling this function.

Parameters
wthe width of the image in pixel
hthe height of the image in pixel
Returns
ErrorCode::NO_ERROR on success, the error code otherwise

Example

#include <AstUtils.h>
int main()
{
// Create a new image using Full HD resolution.
createImage(1920, 1080);
// TODO draw something, store the image.
return 0;
}
