◆ Polygon2d

using astu::Polygon2d = typedef astu::Polygon<double>

Type alias for Polygon template using double as data type.

Polygon are ony defined in two-dimensional space, hence the 2 in this alias is redundant. However, it fits to other naming conventions for classes that do exist in other dimensions on the one hand and on the other hand it helps to separate the d postfix from the class name.
