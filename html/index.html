A collection of utilities for Applied Software Techniques (AST). AST Utilities offer a set of class and functions

This library is intended to be used in combination with the lecture Applied Software Technique (AST) from the bachelor program Media Technology and Design held as well as the lecture Game Archiecture from the master program Interactive Media at the University of Applied Sciences Upper Austria.
